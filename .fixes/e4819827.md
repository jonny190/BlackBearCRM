# Error Fix: No add button for contacts under an accont

# Root Cause Analysis

Based on the repository structure and the reported issue, the problem is likely in the **Account Detail Page** where contacts should be displayed with an option to add new ones. Looking at the codebase structure:

- `packages/client/src/features/accounts/AccountDetailPage.tsx` - Main account detail view
- `packages/client/src/features/contacts/components/ContactList.tsx` - Contact list component
- `packages/client/src/features/contacts/components/ContactForm.tsx` - Contact form component

The root cause is most likely that the `AccountDetailPage.tsx` is missing:
1. An "Add Contact" button in the contacts section
2. Proper state management to handle the add contact modal/form
3. Integration with the ContactForm component
